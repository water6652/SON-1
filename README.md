# MacroPad SON-1
MacroPad SON-1 is a simple seeeduino xiao macropad built for the "HackPad" HackClub project that features:
**1. 2x4 Key matrix**: 2 colums, 4 rows, utilises only 6 pins on the microcontroller.
**2. Rotary encoder with switch**: perfect for audio adjustments or muting audio entirely.
**3. 0.91-inch OLED display**: a resloution of 128x32 pixels, enough to play doom or bad apple.
